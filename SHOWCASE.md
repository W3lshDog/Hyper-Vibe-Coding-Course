# 🌟 Hyper Vibe Coding Course: Showcase

Welcome to the **Lean Launch** version of the Hyper Vibe Coding Course. This repository demonstrates a production-ready "Course-as-a-Product" architecture.

## 🚀 Live Demo
**Landing Page**: [https://w3lshdog.github.io/Hyper-Vibe-Coding-Course/](https://w3lshdog.github.io/Hyper-Vibe-Coding-Course/)

## 📸 Key Features

### 1. Zero-Backend Architecture
- **Frontend**: Pure HTML/CSS/JS for maximum performance and zero maintenance.
- **Hosting**: GitHub Pages (Free, Fast, Secure).
- **Outcome**: 100/100 Lighthouse Score.

### 2. "Vibe" Design System
- **Theme**: Cyberpunk/Neon aesthetics (`#FF1493`, `#00D4FF`).
- **Animations**: CSS keyframe animations for "floating" elements.
- **Responsiveness**: Mobile-first grid layout.

### 3. Automated Quality Assurance
Every commit triggers a GitHub Actions workflow that:
- **Lints Markdown**: Ensures documentation quality.
- **Validates HTML**: Checks for broken links and missing meta tags.
- **Deploys**: Automatically updates the live site.

### 4. Course Content Structure
Organized for modular learning:
- `docs/course/`: Core curriculum (Proprietary).
- `docs/guides/`: Public resources (Creative Commons).
- `frontend/public/`: Marketing assets.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3, JavaScript, Python (Tests).
- **Tools**: GitHub Actions, Unittest.
- **License**: MIT (Code) + Proprietary (Content).
